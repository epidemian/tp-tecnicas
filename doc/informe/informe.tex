\section{Hipotésis}
	\begin{enumerate}
		\item Las máquinas se pueden reparar por separado o se puede reparar toda una línea al mismo tiempo
		\item Se pueden romper tanto las máquinas de control de calidad como las que son 
			exclusivamente para la producción.
		\item Las maquinas pueden levantarse o no con una entrada y una salida determinada. 
			En caso de que no tengan una establecida entonces se les genera una por defecto.
		\item Tanto los precios de la materia prima como los de los productos varían todas las 
			semanas. El resto de los precios se mantiene invariante en el tiempo
		\item Las paredes no pueden ser borradas del terreno adquirido.
	\end{enumerate}
